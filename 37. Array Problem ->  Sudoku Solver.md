# 🧩 Sudoku Solver — Notes

## 📌 Problem Summary

Given a partially filled 9×9 Sudoku board, fill the empty cells (`"."`) so that:

* Each **row** contains digits `1–9` exactly once
* Each **column** contains digits `1–9` exactly once
* Each **3×3 box** contains digits `1–9` exactly once

Board size is **9×9** — classic Sudoku.

---

## 🧠 Core Insight

This is a **constraint satisfaction problem**.

We need to fill cells under strict rules:

```
Valid Row
Valid Column
Valid 3×3 Box
```

There is no greedy shortcut.
The correct approach is **Backtracking** (DFS) with validity checks.

---

## 💡 Backtracking Strategy

1. Find the next empty cell (`"."`)
2. Try digits `'1'` to `'9'`
3. If placing a number keeps the board valid:

   * Place it
   * Recursively solve the rest
4. If no number fits:

   * Backtrack (reset cell)
5. Solution is found when no empty cells remain

This is DFS with **pruning** based on Sudoku rules.

---

## ✔️ Validity Conditions

When placing digit `d` at `(r, c)`:

* `d` must not appear in:

  * same **row**
  * same **column**
  * same **3×3 sub-box**

Box index trick:

```
boxRow = (r // 3)
boxCol = (c // 3)
```

Each box is identified by `(boxRow, boxCol)` pair.

---

## 🧠 Optimization (O1) Lookups

Instead of scanning entire row/column/box for each placement, we maintain:

```
rows[r][digit]
cols[c][digit]
boxes[boxIndex][digit]
```

Where `boxIndex = (r // 3) * 3 + (c // 3)`

This reduces validity checks to **constant time**.

## 🧠 Python Code (LeetCode Style)

```python
from typing import List

class Solution:
    def solveSudoku(self, board):
        rows = [set() for _ in range(9)]
        cols = [set() for _ in range(9)]
        boxes = [set() for _ in range(9)]
        empties = []

        for r in range(9):
            for c in range(9):
                if board[r][c] == ".":
                    empties.append((r, c))
                else:
                    val = board[r][c]
                    rows[r].add(val)
                    cols[c].add(val)
                    boxes[(r // 3) * 3 + (c // 3)].add(val)

        def backtrack():
            if not empties:
                return True

            # Pick cell with minimum options
            min_options = 10
            min_index = -1

            for i, (r, c) in enumerate(empties):
                box = (r // 3) * 3 + (c // 3)
                options = 9 - len(rows[r] | cols[c] | boxes[box])
                if options < min_options:
                    min_options = options
                    min_index = i
                if min_options == 1:
                    break

            r, c = empties.pop(min_index)
            box = (r // 3) * 3 + (c // 3)

            for d in map(str, range(1, 10)):
                if d not in rows[r] and d not in cols[c] and d not in boxes[box]:
                    board[r][c] = d
                    rows[r].add(d)
                    cols[c].add(d)
                    boxes[box].add(d)

                    if backtrack():
                        return True

                    board[r][c] = "."
                    rows[r].remove(d)
                    cols[c].remove(d)
                    boxes[box].remove(d)

            empties.insert(min_index, (r, c))
            return False

        backtrack()```

## ⏱️ Complexity

| Metric | Value                                   |
| ------ | --------------------------------------- |
| Time   | Exponential (worst-case)                |
| Space  | O(1) extra (board is modified in-place) |

Backtracking is inherently exponential, but pruning via row/col/box sets dramatically reduces search space.

## 🧠 Mental Model

Sudoku solving is essentially:

```
DFS + Constraint Checking (row, col, box)
```

Think of it like:

* Explore
* Validate
* Commit
* Undo if needed

This is fundamental in constraint problems like crosswords, n-queens, and logic puzzles.
